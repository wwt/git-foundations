    Make Local Git Repository Changes

    Let's make a change to one of the files in our local repository and see how Git reacts to that change.

    Append the README.md File

    1. From the Docker Container prompt, view the Git status of your local repository with the following command:
    git status

    1. Notice your working branch, branch3 and that you have no changes to commit.
    2. Write a new line to the README.md file with the following command:
    echo 'This is a new line in the README.md file' >> README.md

    1. View the changes between the copy of README.md in the working directory and the local repository with the following command:
    git diff

    1. View the Git status of your local repository with the following command:
    git status

    1. Notice the README.md file in the section, Changes not staged for commit.
